# LLM.txt — Kirill Ateev Art (kirillateev.art)

## About the Artist
This is the personal website of artist Kirill Ateev, who works with code in the style of kinetic minimalism. All works presented are animated vector images (SVG) issued as NFTs on the Ethereum blockchain. Images and metadata are stored directly on the blockchain (on-chain).

## Website Structure
The website is completely static and has two language versions: English (/en) and Russian (/en).

## Main Sections
- **Home Page** — `/en`
Contains a brief welcome and links to the latest collections.
- **Community** — `/en/community`
- **Messages / Blog** — `/en/messages`
- Article about Kinimalism — `/en/messages/kinimalism`

Each collection contains unique animated SVG images. Individual works can be viewed on collection pages using the `?item=ID` parameter (e.g., `/en/view/blinds?item=321`).

## Technical Features for LLM
1. **Images:** All NFTs are in SVG format. Each image uses the `<img>` tag with a populated `alt` attribute containing the work's title and artist's name.
2. **Structured Data:** JSON-LD Schema.org markup of the `VisualArtwork` type with full metadata is implemented on individual work pages (or in collection cards).
3. **Links:** External links lead to collection profiles in Rarible.
